# Healthcare Information Retrieval System

This project builds an Information Retrieval (IR) system for healthcare queries using the MedQuAD dataset.

## Features
- Text preprocessing
- TF-IDF / vectorization
- Query-based retrieval
- Evaluation of IR models

## Dataset
- MedQuAD (medical Q&A dataset)

## Tech Stack
- Python
- Pandas
- Scikit-learn
- NLP techniques

## How to Run
1. Open the notebook in Google Colab
2. Upload dataset (medquad.csv)
3. Run all cells
